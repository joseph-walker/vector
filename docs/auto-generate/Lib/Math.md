# mean

> [Num] -> Num

__Arithemtic mean__

Returns the average of a list, or zero for an empty list.

```
$mean([1, 2, 3]); // (1 + 2 + 3) / 3 = 2
$mean([]); // 0
```

# pow

> Num -> Num -> Num

__Power function__

Arithmetic exponentionation. Raises the second argument to the power
of the first.

```
$pow(2, 3); // 3 ^ 2 = 9
$pow(3, 2); // 2 ^ 3 = 8
```
